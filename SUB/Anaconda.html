Anaconda is one of the python Distribution. It aims to provide everything you need for Data science and Analytics. It includes all the packages which are required. So its best to install Anaconda first. (As a student, I felt Jupyter Notebook is much simpler for starting baby steps in Data Science). You can see the [Anaconda Package list here :] https://docs.anaconda.com/anaconda/packages/pkg-docs

1] Anaconda Download (https://www.anaconda.com/download/#macos)
2] Depending on the OS you are using (I am using MacOS) click on the OS icon( Mac/Windows/Linux)
3] Now you get an option for choosing different version of Python (Say, Python 3.6 and Python 2.7 version). I am going with Python 3.6 version depanding on my needs and packages.
4] Click on the suitable version and wait for it to download.(it takes approximitely 3-4min)
5] Run the downloaded Anaconda package > Next > I Agree > Next > (Set Destination Folder) Next > (Advanced Options: Select Both the options) Install > Next > Finish.
6] Its Done.Search for Anaconda-Navigator in ur System and Open it.


You will see Jupyter Notebook on your Home page of Anaconda Navigator.
1] Click on Launch 2] You can see http://localhost:8888/tree/ has been displayed on you browser
If not you can open a Terminal and type Jupyter Notebook and click Enter. You will see a URL displayed in the output. Copy and paste it in the browser.
(P.S: To check if you have installed a correct version of python, Open a termial and write Python and click Enter)


Now we have installed and successfully able to Launch Jupyter Notebook, Now let's see how to create our first Project in Notebook.
1] Firstly, let's create a folder in the system where we want to store all out Projects( I am storing in, Documents/MyFirstProject)
2] Refresh your browser and go to the selected path (Documents/FirstProject).
3] Under your Localhost -> Files, click on New drop-down at the right-top corner of the page.
4] Select Python 3

Installation is easy and working on Notebook is much simpler and interactive. Anaconda has other python IDE's under it. Explore and see which you find best for analysis.
